2019년도 진행중인 프로젝트 입니다.

1. 스테이지 에디터 구현
스테이지에 대한 정보를 유니티 에디터를 통해서 입력하고 저장할 경우
동적으로 바이너리 파일을 생성해 게임에서 읽어올 수 있도록 구현했습니다.
역으로 파일에 새로운 정보를 쓰는 것도 가능합니다.

2. 스테이지 엑셀 에디터 구현
스테이지의 경우 싱글 컨텐츠이며, 스테이지 시나리오에 등장하는 대사들을
엑셀 형태의 파일로 가지고 있도록 구현했습니다. 따라서 스테이지 정보를
입력할 때 대사 정보들도 입력하도록 하고 있으며, 그러한 정보들은 엑셀 형태로
따로 저장하게 됩니다. 이렇게 구현함으로써 번역가가 외국어를 입력하기에
용이한 형태로 구현했고, 기능적으로도 외국어 정보를 쉽게 세팅할 수 있도록 구현했습니다.
